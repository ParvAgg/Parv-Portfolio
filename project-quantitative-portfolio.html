Quantitative Portfolio Management

Project Overview

This project applied machine learning techniques, such as Random Forest and Principal Component Analysis (PCA), to predict semiconductor stock returns and improve portfolio management strategies during volatile periods.

Tools & Techniques

  • Random Forest for stock return predictions
  • Principal Component Analysis (PCA)
  • Python (scikit-learn, pandas) for machine learning and data analysis

Results

The model achieved a 41% return, outperforming benchmarks and stabilizing risk during market volatility. The PCA reduced dimensionality, improving model accuracy and interpretability.
