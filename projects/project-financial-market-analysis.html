Financial Market Analysis using MATLAB

Project Overview

This project utilized the Fama-French-Carhart and 5-Factor Models in MATLAB to perform comprehensive portfolio analysis. The primary goal was to analyze risk-adjusted returns and optimize portfolio allocation.

Tools & Techniques

  • MATLAB for quantitative analysis
  • Fama-French-Carhart Model
  • 5-Factor Model
  • Time Series Analysis

Results

With an 85% predictive accuracy, this model successfully optimized the risk-return profile of the portfolio, providing critical insights for investment strategy.
