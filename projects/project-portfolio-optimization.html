Investment Portfolio Optimization

Project Overview

This project involved Markowitz Portfolio Optimization using historical price data to determine optimal portfolio allocation with minimized variance.

Tools & Techniques

  • Markowitz Portfolio Theory
  • Historical Price Data Analysis
  • Variance Minimization
  • Python and Excel for data analysis

Results

The optimized portfolio allocation minimized variance, improving risk management strategies. The final allocation yielded a higher Sharpe ratio, demonstrating better returns for the level of risk.
